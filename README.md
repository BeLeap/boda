# Boda

Opinionated replacement of watch.

## Installation

### Using cargo

```bash
cargo install bash
```

### Using homebrew

```bash
brew install beleap/tap/boda
```

### From pre-built binary

Pre-built binary can be found at [release](https://github.com/BeLeap/boda/releases).
